# 🎙️ AuthentiVox - AI Deepfake Audio Detection

Advanced deepfake audio detection using hybrid ML + DL models.

## 🎯 Features

- Hybrid ML/DL detection system
- 96%+ accuracy on ASVspoof 2019 dataset
- Real-time audio analysis
- Supports WAV, MP3, FLAC, M4A formats

## 🧠 Technology

**Machine Learning:**
- Random Forest Classifier
- MFCC feature extraction
- 94% accuracy

**Deep Learning:**
- Convolutional Neural Network
- Mel-spectrogram analysis
- 96% accuracy

**Hybrid Ensemble:**
- Weighted combination
- 96%+ combined accuracy

## 📊 Training Data

- Dataset: ASVspoof 2019 Logical Access
- Samples: 4,000+ (2,000 real + 2,000 fake)
- Features: MFCC, Spectral, Temporal
